# PrixQuartier22

This project is a multi-part application including a landing page, an administrative dashboard, and a user-facing page. It's designed to manage quartier (neighborhood) prices and related data.

## Project Structure

- `admin_dashboard/`: A React-based administrative dashboard for managing various aspects of the application.
- `page_utilisateur/`: Contains the user-facing part of the application. (Details to be added if more information becomes available).
- `landing.html`: The main landing page for the application.
- `assets/`: Contains static assets like CSS and JavaScript files used across the project, primarily for the landing page.

## Getting Started

To get the project up and running, follow these steps.

### Prerequisites

Ensure you have Node.js and npm (Node Package Manager) installed on your system.

### Installation

1.  **Install Root Dependencies (if any)**:
    If there are any root-level dependencies, run:
    ```bash
    npm install
    ```
2.  **Install Admin Dashboard Dependencies**:
    Navigate to the `admin_dashboard` directory and install its dependencies:
    ```bash
    cd admin_dashboard
    npm install
    ```

### Running the Applications

#### Admin Dashboard

To start the development server for the admin dashboard:

```bash
cd admin_dashboard
npm run dev
```
This will typically open the dashboard in your browser at a local address (e.g., `http://localhost:5173`).

#### User Page

(If the `page_utilisateur` contains static HTML, you can open it directly. If it's another application, specific instructions will be needed.)

Navigate to the `page_utilisateur` directory and open its main HTML file (e.g., `index.html`) in your browser, or follow its specific running instructions if available.

#### Landing Page

Open the `landing.html` file directly in your web browser.
