# PrixQuartier22

This project contains a landing page, an admin dashboard, and a user page.

## Getting Started

### Admin Dashboard

Navigate to the `admin_dashboard` directory and follow the instructions in its README.md (if available) or run `npm install` and `npm run dev`.

### User Page

Navigate to the `page_utilisateur` directory and follow the instructions in its README.md (if available).

### Landing Page

Open `landing.html` in your browser.
